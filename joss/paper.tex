\section{Summary}\label{summary}

\texttt{pyGC} is an open-source Python library for estimating Granger
Causality (GC) in the frequency domain from multivariate time-series
data. The package implements both the \emph{parametric} pathway---fitting
a Vector Auto-Regressive (VAR) model via the Yule-Walker equations and
deriving the transfer function analytically---and the
\emph{non-parametric} pathway based on Wilson spectral factorization of a
directly estimated cross-spectral matrix
(\citeproc{ref-wilson1972}{Wilson, 1972};
\citeproc{ref-dhamala2008}{Dhamala et al., 2008}).
An optional covariance-based time-domain GC estimator and a JAX back-end
that JIT-compiles the entire Wilson iteration loop via XLA are also
provided. The library targets neuroscience applications (EEG, MEG, LFP)
but is applicable to any domain where directional information flow
between signals needs to be quantified.

% ============================================================
\section{Statement of Need}\label{statement-of-need}

Granger Causality (\citeproc{ref-granger1969}{Granger, 1969}) is a
standard tool for inferring directed connectivity from neural time
series. Frequency-domain formulations
(\citeproc{ref-geweke1982}{Geweke, 1982};
\citeproc{ref-dhamala2008}{Dhamala et al., 2008}) are particularly
popular because they reveal the frequency band at which the influence
operates, a detail lost in time-domain summaries.

While individual routines exist in MATLAB toolboxes such as MVGC
(\citeproc{ref-barnett2014}{Barnett \& Seth, 2014}) and in scattered
Python snippets, a cohesive, tested, and pip-installable Python library
that covers both estimation pathways, conditional GC, covariance-based
GC, synthetic benchmark models, and GPU-accelerated computation has been
lacking. \texttt{pyGC} fills this gap with a clean NumPy/SciPy API, a
full pytest suite, and optional JAX acceleration.

\section{Background}\label{background}

\subsection{Granger Causality in the Frequency Domain}\label{gc-background}

Given a stationary multivariate process \(\mathbf{X}(t)\), its VAR
representation of order \(m\) is
\begin{equation}
  \mathbf{X}(t) = \sum_{k=1}^{m} \mathbf{A}_k \mathbf{X}(t-k) +
  \boldsymbol{\varepsilon}(t),
  \qquad
  \boldsymbol{\varepsilon}(t) \sim \mathcal{N}(\mathbf{0}, \boldsymbol{\Sigma}),
\end{equation}
where \(\mathbf{A}_k\) are the coefficient matrices and
\(\boldsymbol{\Sigma}\) is the noise covariance. The transfer function is
\(\mathbf{H}(f) = \bigl[\mathbf{I} - \sum_k \mathbf{A}_k^{\top}
e^{-2\pi i k f / F_s}\bigr]^{-1}\) and the cross-spectral matrix is
\(\mathbf{S}(f) = \mathbf{H}(f)\,\boldsymbol{\Sigma}\,\mathbf{H}^*(f)\).

For a bivariate system \((X, Y)\) the pairwise spectral GC measures are
(\citeproc{ref-geweke1982}{Geweke, 1982};
\citeproc{ref-lima2020}{Lima et al., 2020}):
\begin{align}
  I_{X \to Y}(f) &= \ln \frac{S_{YY}(f)}
    {\tilde{H}_{YY}(f)\,\Sigma_{YY}\,\tilde{H}_{YY}^*(f)}, \\
  I_{Y \to X}(f) &= \ln \frac{S_{XX}(f)}
    {H_{XX}^{\circ}(f)\,\Sigma_{XX}\,{H_{XX}^{\circ}}^*(f)},
\end{align}
where \(\tilde{H}_{YY} = H_{YY} + (\Sigma_{YX}/\Sigma_{XX})\,H_{XY}\)
and \(H_{XX}^{\circ} = H_{XX} + (\Sigma_{XY}/\Sigma_{YY})\,H_{YX}\)
are the intrinsic transfer functions after absorbing off-diagonal noise
correlations.

\subsection{Wilson Spectral Factorization}\label{wilson}

When VAR model fitting is not desired, the transfer function and noise
covariance can be recovered directly from a non-parametric estimate of
\(\mathbf{S}(f)\) via the Wilson spectral factorization
(\citeproc{ref-wilson1972}{Wilson, 1972}). The algorithm iteratively
finds an analytic function \(\boldsymbol{\Psi}(f)\) such that
\(\mathbf{S}(f) = \boldsymbol{\Psi}(f)\,\boldsymbol{\Psi}^*(f)\),
yielding \(\mathbf{H}(f)\) and \(\boldsymbol{\Sigma}\) without
assuming a finite-order VAR model.

\subsection{Conditional and Covariance-Based GC}\label{conditional}

For multivariate (\(p > 2\)) systems, \texttt{pyGC} provides conditional
GC (\citeproc{ref-geweke1984}{Geweke, 1984}), which removes shared
driving by conditioning on all remaining channels. Separate Wilson
factorizations are run on each \((p{-}1)\)-dimensional reduced model;
the results are assembled into a \(p \times p\) GC matrix. These
reduced-model factorizations are embarrassingly parallel and are executed
with \texttt{joblib} when \texttt{n\_jobs > 1}.

A covariance-based time-domain GC estimator
(\citeproc{ref-vinck2015}{Vinck et al., 2015}) is also included via
\texttt{covgc\_time}, which computes pairwise directional influences from
sliding-window covariance matrices without an explicit spectral
transform.

\section{Package Structure}\label{package-structure}

\texttt{pyGC} is organised as a single installable package
(\texttt{pygc}) with the following modules:

\begin{itemize}
\tightlist
\item
  \texttt{parametric} --- Yule-Walker VAR fitting
  (\texttt{YuleWalker}) and transfer-function computation
  (\texttt{compute\_transfer\_function}).
\item
  \texttt{non\_parametric} --- Wilson spectral factorization
  (\texttt{wilson\_factorization}).
\item
  \texttt{granger} --- bivariate GC (\texttt{granger\_causality}),
  conditional time-domain GC
  (\texttt{conditional\_granger\_causality}), and conditional spectral
  GC (\texttt{conditional\_spec\_granger\_causality}).
\item
  \texttt{covgc} --- covariance-based time-domain GC
  (\texttt{covgc\_time}).
\item
  \texttt{tools} --- cross-correlation (\texttt{xcorr}), demeaning, and
  regularised determinant (\texttt{rdet}).
\item
  \texttt{ar\_model} --- synthetic benchmark processes: the
  two-variable AR model of \citeproc{ref-dhamala2008}{Dhamala et al.\
  (2008)} and the five-variable model of
  \citeproc{ref-baccala2001}{Baccalá \& Sameshima (2001)}.
\item
  \texttt{spectral\_analysis} --- spectral estimation helpers
  (Fourier CSD, Morlet wavelets, Welch power spectrum) built on
  MNE-Python (\citeproc{ref-gramfort2013}{Gramfort et al., 2013}).
\item
  \texttt{\_jax\_backend} --- optional JAX/XLA back-end with a
  JIT-compiled Wilson loop for CPU/GPU acceleration.
\end{itemize}

\section{Implementation Details}\label{implementation}

\subsection{Vectorised Wilson Factorization}\label{vectorised-wilson}

The reference implementation of Wilson factorization iterates over
frequency bins in Python, which is prohibitively slow for large spectral
matrices or high-resolution frequency axes. \texttt{pyGC} replaces every
inner loop with batch NumPy operations: \texttt{np.linalg.inv} is
applied to the full \((2N \times m \times m)\) tensor in a single call,
and the Plus operator is implemented via \texttt{np.fft.fft}/\texttt{ifft}
along the frequency axis. This yields a roughly \(16\times\) wall-clock
speedup over the loop-based reference on a modern CPU.

\subsection{JAX Back-End}\label{jax-backend}

When JAX (\citeproc{ref-jax2018}{Bradbury et al., 2018}) is installed,
\texttt{wilson\_factorization\_jax} exposes a JIT-compiled version of
the entire Wilson loop via \texttt{jax.lax.while\_loop}. The convergence
condition is encoded as a JAX boolean predicate so that the compiled
kernel exits as soon as the matrix 1-norm drops below \texttt{tol}
without returning to Python between iterations. CPU and GPU execution
are both supported and are selected automatically by the JAX device
back-end.

\subsection{Parallelism}\label{parallelism}

Conditional (spectral) GC requires one Wilson factorization per channel
(reduced models). These are independent and are dispatched via
\texttt{joblib.Parallel} with a thread pool
(\texttt{prefer=\textquotesingle{}threads\textquotesingle{}}), which
avoids serialisation overhead for NumPy-heavy workloads.

\section{Testing}\label{testing}

\texttt{pyGC} ships with a pytest suite of 43 tests covering:

\begin{itemize}
\tightlist
\item Correctness of VAR fitting (Yule-Walker residuals).
\item Spectral factorization convergence and reconstruction error.
\item GC direction recovery on the Dhamala and Baccalá benchmark
  models (\(I_{Y \to X} > I_{X \to Y}\)).
\item Conditional GC matrix sparsity on the Baccalá five-variable
  model.
\item Numerical stability of \texttt{covgc\_time} for various window
  and lag parameters.
\item API consistency between the NumPy and JAX back-ends when JAX is
  available.
\end{itemize}

Tests are run with \texttt{pytest} and a coverage report is generated
via \texttt{pytest-cov}.

\section{Related Software}\label{related-software}

\textbf{MVGC} (\citeproc{ref-barnett2014}{Barnett \& Seth, 2014}) is a
comprehensive MATLAB toolbox for GC analysis but does not provide a
Python interface. \textbf{MNE-Connectivity} provides spectral
connectivity measures in Python but does not implement Wilson
factorization or conditional spectral GC. \textbf{nitime} offers
VAR-based GC but has not been actively maintained. \texttt{pyGC}
complements these tools by providing a modern, tested, pip-installable
Python library that covers the full non-parametric pipeline with GPU
acceleration.
